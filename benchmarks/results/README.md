## Regenerating

**Current tree (recommended — higher repeats for stable medians):**

```bash
uv run python benchmarks/benchmark_router_synthetic.py \
  --repeats 41 --warmup 12 \
  --output-json benchmarks/results/router_run_latest.json
```

**Compare against a previous release:**

```bash
./benchmarks/run_router_compare_0_0_41.sh
# or, for any tag:
# edit the script or pass ALBUCORE_041_WORKTREE=/path/to/worktree
```

**Compare two existing JSONs:**

```bash
uv run python benchmarks/compare_router_json.py \
  benchmarks/results/router_run_latest.json \
  benchmarks/results/router_synthetic_0.0.41.json \
  benchmarks/results/REPORT_<new>_vs_<old>.md
```
